#  Django Blog Application

## Overview

Django Blog Application is a web-based blogging platform developed using Python and Django. The application allows users to create, manage, search, and share blog posts online. Users can register, log in, write blogs, edit or delete their posts, search for blogs, and add comments.

## Features

### User Features

* User Registration & Login
* Create Blog Posts
* Edit Blog Posts
* Delete Blog Posts
* Search Blog Posts
* View Blog Details
* Add Comments

### Admin Features

* Manage Users
* Manage Blog Posts
* Manage Comments

## Technology Stack

* Backend: Python, Django
* Frontend: HTML, CSS, Bootstrap
* Database: SQLite
  
## Installation

```bash
git clone <repository-url>

cd blog_project

pip install -r requirements.txt

python manage.py migrate

python manage.py createsuperuser

python manage.py runserver
```
